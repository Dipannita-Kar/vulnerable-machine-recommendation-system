# Vulnerable Machine Recommendation System

This project aims to build a machine learning-based recommendation system for vulnerable machines.

The system recommends suitable machines based on:
- Difficulty level
- Required skills
- Skills to learn
- Attack category
- Vulnerability type
- Operating system
- Entry point
