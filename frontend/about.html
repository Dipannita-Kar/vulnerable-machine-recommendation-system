About CyberPath

Understanding the recommendation system and how difficulty is determined

What is CyberPath?

CyberPath is an vulnerable machine recommendation system designed for cybersecurity students and practitioners. It helps users find the most suitable vulnerable machines to practice their penetration testing and ethical hacking skills based on their current skill level, preferred attack techniques and learning goals.

Instead of randomly picking a machine to practice on, CyberPath analyses your preferences and recommends machines that best match what you want to learn — saving time and ensuring a more structured learning experience.

CyberPath also includes Machine difficulty Predictor that uses the trained classifier to predict the difficulty level of any machine from VulnHub dataset. Students can search for a machine by name and instantly see the predicted difficulty along with the machine informatin and a direct link to VulnHub machine website.

Machine Dataset

All vulnerable machines currently available in CyberPath are sourced from VulnHub — a platform that provides freely available vulnerable virtual machines for cybersecurity practice. The current dataset contains 305 machines covering a wide range of attack categories, difficulty levels and learning objectives.

Each machine in the dataset has been carefully analysed and labelled with:

Attack Category Difficulty Level Learning Objectives Required Skills Vulnerability Chain Estimated Time Entry Point Attack Path Length

How Difficulty is Determined

Difficulty classification in CyberPath is handled by a trained Random Forest machine learning model that achieves 90.16% accuracy on predicting machine difficulty.

The model considers multiple factors to determine difficulty including the number of attack steps required, the complexity of techniques involved, the estimated time to complete and the types of vulnerabilities present.

Easy

Straightforward attack paths with common techniques. Suitable for beginners learning the basics of penetration testing.

Medium

Multiple attack steps requiring intermediate knowledge. Suitable for students with some experience in CTF challenges.

Hard

Complex attack chains involving advanced techniques. Suitable for experienced practitioners looking for a challenge.

⚠️ Difficulty is Relative

It is important to understand that difficulty is not absolute : it is relative to each individual's experience and background.

A machine labelled as Easy may feel challenging to a complete beginner but straightforward to someone with prior CTF experience. Similarly, a Hard machine may be manageable for someone who specialises in that particular attack category.

CyberPath's difficulty labels are based on the technical complexity of the machine's attack path and the number of steps required : not on subjective human experience. We recommend using the difficulty label as a guide, not a strict rule. If a recommended machine feels too easy or too hard, adjust your difficulty preference and search again.
